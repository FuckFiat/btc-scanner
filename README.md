# 🔍 BTC Scanner

**Исследовательский инструмент для анализа неактивных Bitcoin-адресов**

## 📊 Возможности

| Функция | Описание |
|---------|----------|
| **Сканер** | Поиск неактивных адресов по критериям |
| **Поиск** | Проверка конкретного адреса |
| **Неактивные** | Список адресов 5+ лет неактивности |
| **Киты** | Адреса с балансом 1000+ BTC |
| **Потерянные** | Адреса 10+ лет или потерянные ключи |

## ⚠️ Дисклеймер

Этот инструмент предназначен **только для исследовательских целей**.

- Данные взяты из публичных блокчейн-эксплореров
- Неактивность не означает потерянность
- Никакие приватные ключи не хранятся
- Не пытайтесь получить доступ к чужим средствам

## 🔗 API

- **Blockstream API** - публичные данные блокчейна
- **Mempool.space API** - резервный источник
